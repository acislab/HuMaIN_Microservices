OCRopy Segmentation Micro-Service

GOAL

Extract the single-line images from an binarized image.

USAGE

1. Call OCRopy segmentation micro-service with url: "http://{IP}:{PORT}/segmentationapi".

2. Input:

1) The path of an binarized image, uploaded in form style.
2) Optional. Specified parameter valuses which are listed in the following part, and send them in form style.

3. Output:

A zip file contains the extracted single-line images, and usually they are named as {imagename}_{line NO.}.png

4. Example:

1) Using script "call_seg.py", with command "python call_seg.py -h" to check how to use it. Remenber to change the IP and PORT inside the script.
2) Using curl command. E.g. curl -F "image=@path_to_image" -F "threshold=0.5" -o name_of_output_file http://{IP}:{PORT}/segmentationapi

PARAMETERS (#14)

Parameter Type Default value Comment
minscale Float 1.0 minimum scale permitted
maxlines Integer 300 maximum # lines permitted
scale Float 0.0 the basic scale of the document (roughly, xheight) 0=automatic
hscale Float 1.0 non-standard scaling of horizontal parameters
vscale Float 1.0 non-standard scaling of vertical parameters") # line parameters
threshold Float 0.2 baseline threshold
noise Integer 8 noise threshold for removing small components from lines
usegause Boolean True use gaussian instead of uniform
maxseps Integer 2 maximum # black column separators
sepwiden Integer 10 widen black separators (to account for warping)
maxcolseps Integer 3 maximum # whitespace column separators
csminheight Float 10.0 minimum column height (units=scale)
pad Integer 3 adding for extracted lines
expand Integer 3 expand mask for grayscale extraction

NOTE

1) For curl command, the uploaded image must be an binarized image, and with name/key 'image'.
2) The specified parameters' name must be same with names listed in the above table.
