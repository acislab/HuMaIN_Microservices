Tesseract OCR Micro-Service


GOAL

Extract text from an image.

USAGE

1. Call Tesseract OCR micro-service with url: "http://{IP}:{PORT}/tesseractapi".

2. Input:

1) The image path, uploaded in form style.
2) Optional. Language(s) involved in the image.

3. Output:

An text file related to the image.

4. Example:

1) Using script "call_tesseract.py", with command "python call_tesseract.py -h" to check how to use it. Remenber to change the IP and PORT inside the script.
2) Using curl command. E.g. curl -F "image=@path_to_image" -F "lang=eng" -o name_of_output_file http://{IP}:{PORT}/tesseractapi

LANGUAGES

Parameter Type Default value Supported Language Values Comment
lang String eng eng, chi_tra, chi_sim, osd, equ Support multiple languages. Ex. lang=eng+chi_sim

NOTE

1) The uploaded image's name/key must be 'image'.

2) The value of parameter 'lang' must be same with the supported valued in the above table. For multiple languages, must be combined with symble '+'
